Contents:
Following are the html files, CSS files and javascript files created:

  • index.html: It is the main page of this website.
  • aboutdataset.html: It gives a brief description about the dataset. It also contains five different samples.
    • sample1.html: It contains images of first sample of dataset.
    • sample2.html: It contains images of second sample of dataset.
    • sample3.html: It contains images of third sample of dataset.
    • sample4.html: It contains images of fourth sample of dataset.
    • sample5.html: It contains images of fifth sample of dataset.
  • getdataset.html: It describes the method for participation. It contains a form by which the user can download the dataset.
  • signup.html: It is the registration page of the website. Users can fill up the form and upload their work to register for this event.
    • termsconditions.html: It contains the terms and conditions of this event.
  • schedule.html: It contains the schedule which the event will follow.
  • news.html: It helps users in keeping themselves informed of all that is happening in this event.
  • results.html: It coonsists of names and ranks of top 6 participants.
  • contactus.html: It contains contact information of the event organizers.
  • style.css: It contains the details regarding the website's appearance. I used some parts of the css file I used for assignment 1. Rest of the elements, I saw some youtube videos on those elements and implemented them
  • myjavascript.js: It cotains functions for form validation and congratulatory message alert.
